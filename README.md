# Event-Register-Website
vent Registration Website is a beginner-friendly web application built using Python (Flask), HTML, and SQLite. The website allows users to view available events and register for them by entering their details. All registrations are stored in the SQLite database.
This project demonstrates skills in:

Web development (HTML + CSS + Flask)
Backend application design
Database operations (CRUD)
Routing and form handling
Full-stack application flow

⭐ Features
✔ 1. Home Page – List of Events
Users can see all available events along with basic details like:
Event name
Date
Register button (link to registration form)

✔ 2. Registration Form
Users can enter:
Name
Email
Their details are stored securely in a SQLite database.

✔ 3. Database Integration (SQLite)
A database file (events.db) stores:
Event details
User registrations
The database is created automatically using init_db.py.
✔ 4. Backend Using Flask  
Flask handles:
Routing (/, /register/<id>)
Form submissions
Database connections
Success responses
